### Impala
Веб-приложение, которое позволяет встроить статичный 3D объект в видео с помощью Gaussian Splatting.

---

# Описание

Gaussian Splatting это метод реконструкции 3D сцены из набора изображений или видео. Отличие от классического 3D в том, что сцена представляется как облако трёхмерных примитивов (гауссианов), хранящих данные о своём сегменте. 

Это позволяет учитывать геометрию сцены и её пространственную структуру при рендеринге, что делает интеграцию внешнего 3D-объекта визуально более правдоподобной.

В Impala процесс встраивания объекта ~~будет~~ выглядеть так:

1. Генерация сцены из исходного 2D-изображения или серии изображений (Или загрузка уже готовой сцены)
2. Импорт готовой статичной 3D модели на сцену и её позиционирование, автогенерация shadow-catcher'а
3. Экспорт проекта слиянием 3D сцены и исходного материала.

### Проблемы

- Gaussian Splatting предназначен для реконструкции статичных сцен. Обработка анимированных объектов требует дополнительной обработки.
- Для локальной генерации сцены с нуля требуется значительные объёмы VRAM (8Гб)
